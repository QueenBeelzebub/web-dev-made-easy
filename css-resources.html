What is CSS?

Cascading Style Sheets, or CSS for short, is an integral part of Web Development. With HTMl convering the content of a webpage, CSS will tell your browser how to display that content. This can be anything from fonts to colours to buttons. CSS also covers how to display that information if a user opens the page on a different device. What if the user is using a mobile phone? A monitor? A widescreen? CSS handles all that information. By learning your way around CSS, you will have the ability to great elegant designs with ease.
