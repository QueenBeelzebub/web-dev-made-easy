Get Started

The Journey

When creating a website, there are often a few steps that the web designer takes

The Wireframe

The idea of the website

Before the first line of code, a web designer usually makes a wireframe, which is a mock version of the website, often drawn on paper. It details the layout and flow of a website, and makes it easier to visualise what the website will look like in the end. Here is where the designer also takes into account any accessibility requirements the website might need

The HTML

The skeleton of the website

Now that the layout has been made, the website can be made just functional by the use of HTML. HTML, or Hyper-Text Markup Language, is responsible for the words on the page, and labels what each piece of text is. It is also responsible for making links to other pages. Next comes the style of the page

The CSS

Adding some style

After building the skeleton of the website, we can add some style to make the website easier to digest and look prettier. CSS, short for Cascading Style Sheets, allows the developer to create classes for items in HTML, and apply styles to each item in the class. CSS manages aspects like colour, background colour, alignment, and managing how the site responds to different window sizes. In most cases, you can consider the website done here, but we are going to add some magic to the site.

The Javascript

A touch of Magic

The final step in the visual aspect of a website is the Javascript. Javascript is a programming language that has many features, but one that is used a lot in web development is making the website more interactive or animated. For example, Javascript can be used to make aspects of the website do something on mouseover, or when it's clicked.
Javascript also contains frameworks that can give it additional behaviours like managing databases, or making interactivity easier.

The Backend

Data, data, data

Sometimes called the backend, this invisible feature allows the website to remember its users. Whenever you log into a website, or accept cookies, the website is storing your information on a database, and languages such as PHP (PHP: Hypertext Preprocessor) can manage that data on servers. Most web developers are either front-end (HTML, CSS, Javascript) or back-end (PHP, Python, Ruby, Java). But some developers can do both, they are called full-stack developers.
